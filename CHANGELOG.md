# Changelog

## [0.1.0] - 2026-03-11

Initial public release candidate.

Added:

- issuer metadata URL derivation
- credential configuration, offer, nonce, request, and response helpers
- credential request parsing with stable error atoms
- JOSE JWT proof inspection, claim validation, signature verification, and constrained holder binding
- deferred credential request parsing and deferred response construction
- notification request and notification error response helpers
- parity harness against `oid4vc-ts`
- secondary semantic-oracle documentation for `spruceid/ssi`
